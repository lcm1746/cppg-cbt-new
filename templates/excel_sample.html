CPPG 문제 샘플 엑셀 파일

엑셀 파일 형식에 맞춰 문제를 작성할 수 있도록 샘플 파일을 제공합니다.

📋 사용 방법

  1. 아래 "샘플 파일 다운로드" 버튼을 클릭하여 엑셀 파일을 다운로드합니다.
  2. 다운로드한 파일을 엑셀에서 열어서 문제를 작성합니다.
  3. 컬럼명은 정확히 맞춰주세요 (번호, 문제, 보기1~4, 정답, 해설).
  4. 작성 완료 후 저장하고 웹사이트에 업로드합니다.
